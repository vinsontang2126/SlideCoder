# Function Name add_title

# Function Parameters and Descriptions
- **slide**: Slide object to which the title will be added.
- **text** (`str`): The text content of the title.
- **font_size** (`int`, optional, default: `44`): Font size in points.
- **font_color** (`tuple[int, int, int]`, optional, default: `(0, 0, 0)`): Font color in RGB format, e.g., `(0, 0, 0)` for black.
- **background_color** (`tuple[int, int, int]`, optional, default: `None`): Background color of the textbox in RGB format, e.g., `(255, 255, 255)` for white. If `None`, the background is transparent.
- **position** (`tuple[float, float]`, optional, default: `(1.0, 0.5)`): Position of the textbox (left, top) in inches.
- **size** (`tuple[float, float]`, optional, default: `(8.0, 1.0)`): Size of the textbox (width, height) in inches.
- **alignment** (optional, default: `PP_ALIGN.LEFT`): Horizontal alignment of the text (e.g., `PP_ALIGN.LEFT`, `PP_ALIGN.CENTER`, `PP_ALIGN.RIGHT`).
- **vertical_alignment** (optional, default: `MSO_VERTICAL_ANCHOR.TOP`): Vertical alignment of the text (e.g., `MSO_VERTICAL_ANCHOR.TOP`, `MSO_VERTICAL_ANCHOR.MIDDLE`, `MSO_VERTICAL_ANCHOR.BOTTOM`).
- **space_before** (optional, default: `None`): Space before the paragraph in points.
- **space_after** (optional, default: `None`): Space after the paragraph in points.
- **line_spacing** (optional, default: `None`): Line spacing for the paragraph.

# Function Return Value
- **slide**: The slide object with the added title textbox.


# Function Name add_rectangle

# Function Parameters and Descriptions
- **slide**: Slide object to which the rectangle will be added.
- **coords** (`list[float]`): The position and size of the rectangle, in the format `[left, top, width, height]`.
- **line_color** (`tuple[int, int, int]`, optional, default: `(0, 0, 0)`): Line color in RGB format, e.g., `(0, 0, 0)` for black.
- **line_width** (`int`, optional, default: `1`): Line width in points.
- **fill_color** (`tuple[int, int, int]`, optional, default: `None`): Fill color in RGB format, e.g., `(255, 255, 255)` for white. If `None`, no fill is applied.

# Function Return Value
- **slide**: The slide object with the added rectangle.

# Function Name add_text_box

# Function Parameters and Descriptions
- **slide**: Slide object to which the text box will be added.
- **text** (`str`): The text content to be added to the text box.
- **coords** (`list[float]`): Position and size of the text box in the format `[left, top, width, height]`, specified in inches.
- **font_size** (`int`, optional, default: `18`): Font size in points.
- **font_color** (`tuple[int, int, int]`, optional, default: `(0, 0, 0)`): Font color in RGB format, e.g., `(0, 0, 0)` for black.
- **background_color** (`tuple[int, int, int]`, optional, default: `None`): Background color of the text box in RGB format, e.g., `(255, 255, 255)` for white. If `None`, the background is transparent.
- **alignment** (`int`, optional, default: `PP_ALIGN.LEFT`): Horizontal alignment of the text (e.g., `PP_ALIGN.LEFT`, `PP_ALIGN.CENTER`, `PP_ALIGN.RIGHT`).
- **vertical_alignment** (`int`, optional, default: `MSO_VERTICAL_ANCHOR.TOP`): Vertical alignment of the text (e.g., `MSO_VERTICAL_ANCHOR.TOP`, `MSO_VERTICAL_ANCHOR.MIDDLE`, `MSO_VERTICAL_ANCHOR.BOTTOM`).
- **space_before** (`float`, optional, default: `None`): Space before the paragraph in points.
- **space_after** (`float`, optional, default: `None`): Space after the paragraph in points.
- **line_spacing** (`float`, optional, default: `None`): Line spacing for the paragraph. If a large integer (>100) is provided, it is converted to 1.5. Otherwise, values are clamped between 0.0 and 132.0.

# Function Return Value
- **slide**: The slide object with the added text box.


# Function Name add_freeform

# Function Parameters and Descriptions
- **slide**: Slide object to which the free-form shape will be added.

- **coords** (`list[float]`): Position and size of the free-form shape in the format `[left, top, width, height]`, specified in inches.
- **background_color** (`tuple[int, int, int]`, optional, default: `None`): Background color of the free-form shape in RGB format, e.g., `(255, 255, 255)` for white. If `None`, no background color is applied.

# Function Return Value
- **slide**: The slide object with the added free-form shape.


# Function Name add_text

# Function Parameters and Descriptions
- **slide**: Slide object to which the text will be added.
- **text** (`str`): The text content to be added.
- **coords** (`list[float]`): Position and size of the text box in the format `[left, top, width, height]`, specified in inches.
- **font_size** (`int`, optional, default: `20`): Font size in points.
- **bold** (`bool`, optional, default: `False`): Whether the text is bold.
- **color** (`tuple[int, int, int]`, optional, default: `(0, 0, 0)`): Font color in RGB format, e.g., `(0, 0, 0)` for black.
- **background_color** (`tuple[int, int, int]`, optional, default: `None`): Background color of the text box in RGB format, e.g., `(255, 255, 255)` for white. If `None`, the background is transparent.
- **auto_size** (`bool`, optional, default: `True`): Whether to automatically resize the text box (Note: Function sets `MSO_AUTO_SIZE.NONE`, ignoring this parameter).
- **alignment** (`int`, optional, default: `PP_ALIGN.LEFT`): Horizontal alignment of the text (e.g., `PP_ALIGN.LEFT`, `PP_ALIGN.CENTER`, `PP_ALIGN.RIGHT`).
- **vertical_alignment** (`int`, optional, default: `MSO_VERTICAL_ANCHOR.TOP`): Vertical alignment of the text (e.g., `MSO_VERTICAL_ANCHOR.TOP`, `MSO_VERTICAL_ANCHOR.MIDDLE`, `MSO_VERTICAL_ANCHOR.BOTTOM`).
- **space_before** (`float`, optional, default: `None`): Space before the paragraph in points.
- **space_after** (`float`, optional, default: `None`): Space after the paragraph in points.
- **line_spacing** (`float`, optional, default: `None`): Line spacing for the paragraph.

# Function Return Value
- **slide**: The slide object with the added text.
